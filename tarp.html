타프 설치 방법

장비 확인부터 설치 완료까지 실제 작업 흐름에 맞춘 타프 설치 순서입니다.

01

장비 구성 확인

타프 천, 폴대, 스트링, 스토퍼, 팩, 망치를 먼저 한곳에 펼쳐 빠진 장비가 없는지 확인합니다.

02

바람 방향 확인

바람을 넓은 면으로 직접 받지 않도록 방향을 잡고, 주변 공간과 팩을 박을 위치를 확인합니다.

03

타프를 펼치고 중심선 맞추기

타프 천을 바닥에 넓게 펼친 뒤 메인 폴대가 들어갈 중심선과 각 모서리 위치를 맞춥니다.

04

메인 폴대 세우기

메인 폴대를 타프 고리에 끼우고 스트링을 걸어 폴대가 수직으로 설 수 있게 잡아줍니다.

05

팩 박고 스트링 고정

스트링을 45도 방향으로 당겨 팩을 박고, 스토퍼로 장력을 조절해 천이 처지지 않게 고정합니다.

06

설치 완료 확인

전체 폴대와 스트링 장력을 다시 확인하고, 빗물이 흐를 방향과 그늘 면적이 적절한지 마지막으로 점검합니다.

설치 체크포인트

  • 팩은 지면과 45도 각도로 박고, 스트링이 당겨지는 반대 방향으로 기울입니다.
  • 바람이 강할 때는 메인 폴대 주변 스트링을 먼저 고정한 뒤 사이드 폴대를 세웁니다.
  • 철수 전에는 스트링과 팩을 먼저 모아 분실을 막고, 젖은 타프는 완전히 말린 뒤 보관합니다.
